# Datos meteorológicos de la estación NH0437

Dataset con observaciones diarias de temperatura (150 cm de abrigo) y
fecha correspondientes a la estación NH0437.

## Usage

``` r
NH0437
```

## Format

Un data frame con las columnas:

- id:

  Identificador de la estación (carácter).

- fecha:

  Fecha de la medición (Date).

- temperatura_abrigo_150cm:

  Temperatura registrada a 150 cm, en °C (numérico).
